Add Page Config

We can use YAML format at top of every page to set config. Separate config and content by 3 dashes (---).

For Example:

layout: html.twig
title: My Site
flower:
    sakura: World

---
<html lang="en">
<head>
    <title>{{ config.title }}</title>
</head>
<body>
    Hello {{ config.flower.sakura }}
</body>
</html>

All config data will put in config property.
